# Instalar MS Windows 11 en maquina virtual KVM Linux

Regularmente doy soporte tecnico y acompañamiento a personas Defensoras de Derechos Humanos y periodistas por lo que afortunadamente tengo amplia experiencia con sistemas operativos distintos a mi sistema operativo de cabecera (Debian), por lo que tengo en mi computadora principal varias maquinas virtuales para tener el entorno al que estoy dando soporte, por lo que siempre tengo maquinas virtuales con Windows, ahora que se forzó el cambio a Windows 11 [https://support.microsoft.com/es-es/windows/-puedo-actualizar-a-windows-11-14c25efc-ecb7-4ce6-a3dd-7e2e24476997](https://support.microsoft.com/es-es/windows/-puedo-actualizar-a-windows-11-14c25efc-ecb7-4ce6-a3dd-7e2e24476997)

Entonces me vi en la necesidad de instalarme una maquina virtual de Windows 11, pero para hacer esto se tienen que establecer los siguientes parametros.


HKEY_LOCAL_MACHINE\SYSTEM\Setup\MoSetup

crear DWORD (32 bits)

AllowUpgradesWithUnsupportedTPMOrCPU →  1.

HKEY_LOCAL_MACHINE\SYSTEM\Setup\

Crear nueva clave LabConfig.

dentro de LabConfig, crea DWORD (32-bit):

BypassTPMCheck →  1.
BypassSecureBootCheck → 1.
BypassRAMCheck → 1.

Apaga todas las MV en ejecución y detenga el módulo kvm_probe:

---- sudo modprobe -r kvm_intel

Activa la característica anidada:

sudo modprobe kvm_intel nested=1
